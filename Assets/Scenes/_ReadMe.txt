Pour ajouter une scene contenant un level :
1. Ajouter la scene dans les Build Settings
2. Rajouter le nom de la scene dans le tableau RegisterdLevelnames du scriptable object LevelsDatabase
3. Noter l'index dans l'array de la scene qui vient d'etre ajoutee. Rajouter un bouton dans la scene Selection Niveau et lui filer l'index en argument.
4. Rajouter le bouton à l'array des boutons de l UI Manager
